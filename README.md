# Grail Movement Nigeria — Building Projects

Central repository for the planning, design, construction and supervision of
**Grail Movement Halls of Worship and related buildings in Nigeria**.

## Why this repository exists

Several Halls of Worship exist across Nigeria and more are under construction.
For years these projects were executed without a common build guideline, which
led to:

- unnecessary and unpredictable cost overruns,
- inconsistent quality between sites,
- weak project supervision and control,
- designs and documents that could not be reused between projects.

This repository fixes that by providing **one unified guide** that every
Grail Movement building project in Nigeria must follow, so that projects are
**cost-effective, properly supervised, and held to a consistent standard**.

## How to use this repository

1. **Starting a new project?** Begin with the
   [Project Guidelines](docs/01-project-guidelines.md) and complete the
   [Project Charter template](templates/project-charter.md).
2. **Producing drawings or models?** Follow the
   [BIM Guidelines](docs/02-bim-guidelines.md) before creating any model or
   drawing.
3. **Managing money?** No expenditure outside the procedures in
   [Cost Management](docs/03-cost-management.md).
4. **On site?** Supervisors report using the templates and rules in
   [Quality & Supervision](docs/04-quality-and-supervision.md).
5. **Adding a new hall or updating status?** Edit the
   [Hall Registry](docs/05-hall-registry.md).

## Contributing

Changes to guidelines are made by pull request so that every revision is
reviewed and recorded. Each document carries a version number and revision
history — update it whenever the document changes.
