---

This registry is the index of every Grail Message Foundation building project in
Nigeria. **Every hall gets one row and one project code**, and the code links
to its documentation folder (structure per GMF-BIM-001 §7).

Update this file by pull request whenever a project is added or its status
changes (at minimum at every gate).

## Status codes

| Code | Meaning |
|---|---|
| `PLANNED` | Charter approved (Gate 0), pre-feasibility |
| `FEASIBILITY` | Phase 1 in progress |
| `DESIGN` | Phase 2 in progress |
| `TENDER` | Phase 3 in progress |
| `CONSTRUCTION` | Phase 4 in progress |
| `DEFECTS` | Practically complete, defects liability running |
| `COMPLETE` | Gate 5 closed, in use |
| `ON-HOLD` | Suspended by Board decision (note the reason) |

## Existing halls built before this guideline

Halls completed before this guideline was adopted should still be registered.
For each, the minimum retrofit documentation is:

1. A project code assigned here,
2. A measured survey and **as-built BIM model** (target LOD 300, produced when
   resources allow — prioritise halls likely to be extended or repaired),
3. Land documents and any approvals scanned into `01-Admin/`,
4. Known defects or maintenance needs logged.
